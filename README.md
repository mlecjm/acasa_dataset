# A-CASA Synthetic Dataset
The A-CASA Synthetic Dataset for the *"A-CASA: A Clinically-Grounded NLP Triage Framework with Blockchain Data Assurance for Suicide-Risk Stratification in Unstructured Pre-Hospital 119 Emergency Dispatch Logs''* paper that reports results on 4,218 retrospective 119 emergency dispatch records collected from a metropolitan EMS dispatch centre in Seoul, Republic of Korea (January 2022–December 2024).
